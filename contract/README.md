# 🎨 Digital Art Auction (Soroban Smart Contract)

## 📌 Project Description
This project is a basic decentralized Digital Art Auction smart contract built using Soroban (Stellar smart contracts). It allows users to create an auction, place bids, and finalize the auction on-chain in a transparent and trustless way.

## 🚀 What it does
- Allows an artist or seller to create an auction for digital artwork.
- Users (bidders) can place bids on the artwork.
- The contract tracks the highest bid and highest bidder.
- The seller can end the auction, finalizing the result.

## ✨ Features
- 🧑‍🎨 Auction creation by seller
- 💰 Real-time highest bid tracking
- 🥇 Automatic highest bidder update
- 🔒 Secure auction closing (only seller can end it)
- 📦 On-chain storage of auction state
